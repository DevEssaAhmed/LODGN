## Run Locally

Clone the project

```bash
  git clone https://github.com/DevEssaAhmd/LODGN.git
```

Go to the project directory

```bash
  cd Lodgn
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm  run dev
```
